Personalizar el chat de OBS

Con esta herramienta podrás modificar facilmente el formato del texto del chat. Juega con los diferentes parámetros y obtendrás el código CSS personalizado que posteriormente deberás introducir en la ventana correspondiente de SteamlabsOBS y OBS.

Por defecto, la ventana del chat los mensajes aparecen con una tipografía estándar, en color negro y con fondo transparente. No obstante, las características del texto pueden modificarse para adecuarse al layout de cada programa.

¿Donde pongo el código?

El código generado podeis colocarlo en el apartado "CSS Personalizado" de la fuente de navegador que usais para introducir el chat en OBS. En las siguientes imagenes podeis ver cual es ese apartado tanto en Streamlabs Obs como en OBS Studio.
